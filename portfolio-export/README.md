# Portfolio Professionnel Moderne

Ce dossier contient tous les fichiers de code source du portfolio.

## Structure du projet

```
portfolio-export/
├── src/
│   ├── components/
│   │   ├── Header.tsx
│   │   ├── Hero.tsx
│   │   ├── About.tsx
│   │   ├── Skills.tsx
│   │   ├── Projects.tsx
│   │   ├── Contact.tsx
│   │   └── Footer.tsx
│   ├── App.tsx
│   ├── main.tsx
│   └── index.css
├── index.html
├── package.json
├── tailwind.config.js
├── vite.config.ts
└── tsconfig.json
```

## Technologies utilisées

- **React 18** avec TypeScript
- **Tailwind CSS** pour le styling
- **Lucide React** pour les icônes
- **Vite** comme bundler
- **Supabase** pour la base de données (optionnel)

## Installation

1. Copiez tous les fichiers dans votre projet
2. Installez les dépendances : `npm install`
3. Lancez le serveur de développement : `npm run dev`
